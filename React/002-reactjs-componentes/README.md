### :gear: Colocando para funcionar

```bash
# Clone o repositorio na pasta desejada
$ git clone https://github.com/DanielVII/treino.git
```

```bash
# Acesse a pasta  onde foi feito o clone
$ cd suaPasta/
```

```bash
# Instale as dependências
$ npm install
```

```bash
# Agora é só abrir
$ npm start
```
